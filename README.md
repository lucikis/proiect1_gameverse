Website-ul meu este despre o firma unde se vand console de gaming si jocuri. Pe langa asta firma ofera si suport tehnic atat pentru componentele hardware cat si cele software.

Pagina Servicii prezinta pe scurt ce are de oferit compania: Vanzare de console, jocuri si suport tehnic.

Pagina Preturi are rolul de a prezenta preturile produselor si metodele de plata.

Pagina Contact are rolul de a oferi clientilor o metoda de a contacta firma prin intermediul unui mesaj care poate fi trimis direct de pe website.

Am folosit JavaScript pentru generarea dinamica a tabelului cu preturi si pentru a calcula pretul total pentru produsele din tabel.
